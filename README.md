# holidays V1
API pública básica en GitHub que calcule días laborables en Estados Unidos, usando GitHub Pages + JavaScript (sin necesidad de servidor propio)

✨ Características incluidas
✅ 3 endpoints en uno
✅ Soporte 2020-2030 (fácil de extender)
✅ Demo interactiva completa
✅ Nombres de feriados en español
✅ Manejo de errores robusto
✅ Respuestas JSON consistentes
✅ Cálculo preciso de días hábiles
✅ Lista de feriados dentro del rango
