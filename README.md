# holidays
API pública básica en GitHub que calcule días laborables en Estados Unidos, usando GitHub Pages + JavaScript (sin necesidad de servidor propio)

📌 Características
✅ Completamente gratis (GitHub Pages)
✅ Soporta hasta 2030 (puedes agregar más años)
✅ Retorna JSON automático
✅ Incluye demo interactiva
✅ Feriados federales de EE.UU.
✅ Detecta fines de semana

🔄 Mejoras posibles
Agregar cálculo de días hábiles entre dos fechas

Soporte para más años (consultando API externa de feriados)

Parámetro ?format=json o ?format=xml

Endpoint /api/is-working-day/2025-07-04
